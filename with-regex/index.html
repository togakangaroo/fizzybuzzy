FizzBuzz with a Regex

"Is this decimal string divisible by 3?" is a regular language: a 3-state machine can track the digit sum mod 3 while reading left to right ([0369] stays put, [147] advances one state, [258] advances two). Every finite automaton can be flattened into a regular expression by state elimination — do that, and FizzBuzz becomes a single regex with three named groups: fizzbuzz (divisible by 15), buzz (ends in 0 or 5), and fizz (digit sum divisible by 3). Alternation is ordered, so fizzbuzz gets first refusal, and the trailing \b forces each branch to swallow the whole number or fail.

What the fizz branch is computing

reading digits: [0369] dotted (+0) · [147] solid (+1) · [258] dashed (+2). Divisible by 3 ⇔ the walk ends on the double-ringed state.
