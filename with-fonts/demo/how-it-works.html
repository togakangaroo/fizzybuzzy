How the FizzBuzz font computes

Three GSUB lookups run in order over the glyph buffer. The trick making it a state machine: within a lookup, the backtrack context matches glyphs that have already been substituted — so state written at one position is readable at the next.

1 STATE 2 CLASSIFY 3 BLANK

digit sum ≡ 0 (mod 3) ≡ 1 ≡ 2 word glyph backtrack (already substituted!)
