Wait, what's a ligature?

A ligature is a font swapping two or more letters for one purpose-drawn glyph when it sees them next to each other — usually to fix an ugly collision, like the dot of an "i" crashing into the hook of an "f". Most serif fonts do this silently, all the time. Look at "office" below, set in the same font with ligatures on and off:

That substitution — "when you see this exact letter sequence, draw this other glyph instead" — is a rule in the font file itself (an OpenType GSUB lookup), not something your word processor is doing. This cheat font just points that same mechanism at numbers instead of letters: "when you see the exact digits 1 5, draw the glyph for the word FizzBuzz instead." 47 of those rules, one per qualifying number, is the entire "program."
