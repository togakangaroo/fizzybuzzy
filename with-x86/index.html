FizzBuzz in x86 Machine Code

The program is a 512-byte boot sector. The computer it boots on is v86 — an x86 PC emulated in WebAssembly. No operating system is ever loaded: the BIOS reads the floppy's first sector to address 7C00, jumps to it, and FizzBuzz talks to the BIOS teletype routine directly.
