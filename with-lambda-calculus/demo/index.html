FizzBuzz with Nothing — the lambda calculus edition

Every value below was computed using only single-argument arrow functions: no numbers, no strings, no booleans, no arrays, no operators, no if, no named recursion. A JavaScript translation of Tom Stuart's Programming with Nothing.
