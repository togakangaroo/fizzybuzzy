FizzBuzz with Pure CSS

The markup is 100 empty <li>s. A CSS counter numbers them, then more specific rules override the generated content on every 3rd and 5th child. The cascade is the control flow.

The program

Idea from Lindsor's Pure CSS FizzBuzz codepen; a sibling demo implements the ordered-list variant from Trys Mudford's FizzBuzz in CSS, which composes Fizz + Buzz from an <ol>'s markers and separate ::before/::after pseudo-elements.
