FizzBuzz with an Ordered List

The markup is an <ol> of 100 empty <li>s, so the browser numbers them for free. Three rules finish the job — and "FizzBuzz" is never written anywhere: it composes when a ::before "Fizz" and an ::after "Buzz" land on the same item.

The program

Idea from Trys Mudford's FizzBuzz in CSS; a sibling demo implements the CSS-counter variant from Lindsor's codepen.
