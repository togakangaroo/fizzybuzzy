FizzBuzz Without Modulo

No division, no remainder — just two free-running cycles, one of period 3 and one of period 5. Each tick pulls the next slot from both; concatenate whatever comes out, and if that's the empty string, the number speaks for itself. The two words collide every 15 all on their own. No if statements either.

Bonus variant: Tail recursive, with the branching spelled out 
function* fizzbuzz(n = 1,
                   till3 = cycle([1, 1, 0]),
                   till5 = cycle([1, 1, 1, 1, 0])) {
    if (!till3.next().value)
        if (!till5.next().value) yield "FizzBuzz";
        else yield "Fizz";
    else if (!till5.next().value) yield "Buzz";
    else yield `${n}`;
    yield* fizzbuzz(n + 1, till3, till5);
}
